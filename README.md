# react-native-todo
Very lightweight and smooth design todo app with react-native


## How to install ?
``` bash
# Install necessary packages
yarn install

# Create android&ios file
react-native eject

# Update native files
react-native link

# Run app on emulator
react-native run-android
react-native run-ios
```
